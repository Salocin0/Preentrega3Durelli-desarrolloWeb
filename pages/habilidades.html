Front-end

React

React es una biblioteca de JavaScript para construir interfaces de usuario interactivas y reutilizables. Es ampliamente utilizado para el desarrollo de aplicaciones web de una sola página (SPA) y se basa en componentes que permiten construir interfaces dinámicas y eficientes.

Angular

Angular es un framework de desarrollo de aplicaciones web desarrollado por Google. Utiliza el lenguaje TypeScript y se basa en componentes para construir aplicaciones web robustas y escalables. Angular ofrece muchas características y herramientas para facilitar el desarrollo y el mantenimiento de aplicaciones complejas.

HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar el contenido de las páginas web. Se utiliza para definir la estructura y el contenido de un documento web, como títulos, párrafos, enlaces, imágenes y más.

CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para describir la presentación y el estilo de un documento HTML. Se utiliza para definir cómo se mostrarán los elementos HTML en una página web, incluyendo aspectos como colores, fuentes, diseños y efectos visuales.

Java Script

JavaScript es un lenguaje de programación de alto nivel y orientado a objetos. Es ampliamente utilizado en el desarrollo web para agregar interactividad y dinamismo a las páginas. JavaScript se ejecuta en el navegador del cliente y permite manipular el contenido de la página, responder a eventos, realizar validaciones y mucho más.

Back-end

PostgreSQL

PostgreSQL es un sistema de gestión de bases de datos relacional de código abierto. Es conocido por ser robusto, confiable y escalable, y es ampliamente utilizado en aplicaciones web y empresariales. Ofrece soporte para consultas SQL avanzadas, integridad de datos y características de seguridad.

Java

Java es un lenguaje de programación de propósito general y orientado a objetos. Es conocido por su portabilidad y se utiliza ampliamente en el desarrollo de aplicaciones empresariales y de servidor. Java es altamente escalable y cuenta con una amplia biblioteca estándar y un ecosistema de herramientas y frameworks.

Spring

Spring es un framework de desarrollo de aplicaciones Java que proporciona una infraestructura completa para el desarrollo de aplicaciones empresariales. Ofrece características como inyección de dependencias, gestión transaccional, seguridad, creación de API REST y mucho más, lo que facilita el desarrollo rápido y eficiente de aplicaciones robustas.

Postman

Postman es una herramienta de colaboración y desarrollo de API que permite probar, documentar y compartir API. Es ampliamente utilizado por desarrolladores para enviar solicitudes HTTP, realizar pruebas de integración, capturar y validar respuestas, y colaborar con equipos en el desarrollo de API.

Node js

Node.js es un entorno de ejecución de JavaScript basado en el motor de JavaScript V8 de Chrome. Permite ejecutar JavaScript en el servidor, lo que lo hace adecuado para el desarrollo de aplicaciones web y de red escalables y de alto rendimiento. Node.js también cuenta con un vasto ecosistema de paquetes y frameworks que facilitan el desarrollo de aplicaciones server-side.
